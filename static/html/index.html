Page Directory

Home

HTTP Methods Path Description
GET / Direct to the home page

Users

HTTP Methods Path Description
GET /users View all users in the database
GET /users/:id Retrieve a single user based on their ID
POST /users/register Register a new user to the database
PATCH/PUT /users/update/:id Update an existing user's details
DELETE /users/delete/:id Delete a user

Products

HTTP Methods Path Description
GET /products Retrieve all products
GET /products/:id Fetch a single product
POST /products/addProduct Add a new product to the database
PUT/PATCH /products/update/:id Update an existing product
